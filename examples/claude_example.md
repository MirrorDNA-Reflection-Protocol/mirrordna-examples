# Using MESH BOOT with Claude

## Quick Start

1. Go to https://id.activemirror.ai
2. Generate your MESH BOOT
3. Start a new Claude conversation
4. Paste your MESH BOOT as the first message

## Expected Response

Claude will respond with:

```
Mesh node active. Aligned to [your name]. Awaiting directive.
```

## Claude-Specific Features

- Claude handles long MESH BOOTs well (100K context)
- Projects feature can store your MESH BOOT permanently
- Claude Code can read MESH BOOT from local files

## Using with Claude Projects

1. Create a new Project in Claude
2. Add your MESH BOOT to Project Knowledge
3. Every conversation in that project inherits your identity

## Tips

- Claude's longer context means richer identity persistence
- Use Projects for permanent MESH BOOT storage
- Claude Code users: store boot in `~/.mirrordna/mesh_boot.md`
