# Using MESH BOOT with Gemini

## Quick Start

1. Go to https://id.activemirror.ai
2. Generate your MESH BOOT
3. Start a new Gemini conversation
4. Paste your MESH BOOT as the first message

## Expected Response

Gemini will respond with:

```
Mesh node active. Aligned to [your name]. Awaiting directive.
```

## Gemini-Specific Notes

- Gemini has a large context window (1M+ tokens)
- Works well with detailed MESH BOOTs
- Google AI Studio supports system prompts for permanent boots

## Using with AI Studio

1. Open Google AI Studio
2. Set your MESH BOOT as the System Instruction
3. All prompts will include your identity context

## Tips

- Gemini's multimodal capabilities work with MESH BOOT
- Can reference images/docs while maintaining identity
- Use AI Studio for persistent configuration
